# 🤖 AI Virtual Calling Assistant

An intelligent **AI-powered virtual assistant** that allows users to interact using **voice and text**, providing real-time, context-aware responses using **Natural Language Processing (NLP)** and **Generative AI**.

---

## 📌 Overview

The AI Virtual Calling Assistant is designed to automate communication and enquiry handling. It processes user input (voice or text), understands intent using AI techniques, and generates meaningful responses in real time.

This project demonstrates the practical application of:

* Artificial Intelligence (AI)
* Natural Language Processing (NLP)
* Speech Recognition
* Generative AI

---

## 🚀 Features

* 🎤 Voice-based interaction (Speech-to-Text)
* 💬 Text-based chatbot support

## ⚙️ How It Works

1. User provides input (voice or text)
2. Voice is converted to text (if applicable)
3. Input is processed using NLP
4. AI model generates a response
5. Response is returned as text and voice

## 🛠️ Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/ai-virtual-calling-assistant.git
cd ai-virtual-calling-assistant
```

### 2. Install Dependencies

#### Frontend

```bash
cd frontend
npm install
npm run dev
```

#### Backend

```bash
cd backend
npm install
npm start
```

---

## 🔑 Environment Variables

Create a `.env` file in the backend folder and add:

AI_API_KEY=your_api_key_here
